```
受影响资产价格变动 = 触发源价格变动 × 传导系数 × (1 - 衰减)
衰减 = 延迟tick数 × 0.05（越晚传导衰减越多）
```

## 三、假新闻揭穿概率公式

### 核心公式

```
每tick揭穿概率 = base_rate × (1 + time_factor) × scrutiny × (1 - credibility) × history_penalty
```

### 3.1 各因子定义

**base_rate（基础揭穿率/tick）**

| 投放渠道 | base_rate |
|----------|-----------|
| 社交媒体 | 0.8% |
| 新闻网站 | 1.2% |
| KOL渠道 | 0.6% |
| 暗网 | 0.3% |
| 内部泄露 | 0.5% |

**time_factor（时间累积）**

```
time_factor = 已存活tick数 × 0.03
封顶: 3.0（约100tick/500秒后封顶）
```

**scrutiny（审查强度）**

| 条件 | scrutiny加成 |
|------|-------------|
| 基础值 | 1.0 |
| 该新闻已造成>5%价格波动 | ×1.5 |
| 有KOL公开质疑该新闻 | ×2.0 |
| 有量化先知·林的数据检测 | ×2.5 |
| 目标国处于"信息战"状态 | ×0.5（信息混乱反而掩护） |
| 全球情绪极端（<15或>85） | ×0.7（极端情绪下AI判断力下降） |

**credibility（新闻当前可信度）**

| 来源 | 基础可信度 |
|------|-----------|
| 无加成 | 40% |
| 伪造数据源 | +20% |
| 专家背书 | +40% |
| KOL转发 | +15%/个 |
| 暗影操盘手"欺骗大师"天赋 | +25% |
| 同类新闻近期被揭穿过 | -30% |

可信度95%封顶。

**history_penalty（历史惩罚）**

```
history_penalty = 1 + (本局已揭穿假新闻数 × 0.15)
```

每被揭穿过一次假新闻，后续假新闻揭穿率+15%。

### 3.2 计算示例

```
场景: 社交媒体发布"伽蓝央行即将加息"，带伪造数据源
已存活8个tick，造成6%价格波动，首次撒谎

base_rate = 0.8%
time_factor = 8 × 0.03 = 0.24
scrutiny = 1.0 × 1.5(价格波动>5%) = 1.5
credibility = (40% + 20%) = 60%
history_penalty = 1.0

每tick揭穿概率 = 0.8% × 1.24 × 1.5 × 0.4 × 1.0 = 0.60%/tick
```

### 3.3 AI揭穿行为

| AI类型 | 主动检测概率/tick | 说明 |
|--------|-----------------|------|
| 量化先知·林（如果选了） | 5% | 数据分析能力最强 |
| 鲸鱼AI | 2% | 有研究团队 |
| KOL AI | 3% | 舆论敏感 |
| 韭菜AI | 0.3% | 大部分不会质疑 |
| 骗子AI | 不会揭穿 | 它们自己也在骗 |

AI成功检测后发公开质疑推文，触发scrutiny ×2.0加成，加速揭穿。

### 3.4 揭穿后连锁反应

```
揭穿瞬间:
├── 新闻标记为[已揭穿]，红色删除线
├── 已被该新闻影响的情绪立即反转（恢复50-80%）
├── 玩家公信力 -15~-25（看影响范围和新闻存活时间）
├── 粉丝流失 -10%~-30%（教父角色额外-10%）
├── 关系值：所有KOL AI关系 -5~-15
└── 部分理性AI会反向操作（制造"揭穿反弹"行情）
```

## 四、战争结束条件与后果

### 4.1 战争持续时间基准

| 战争规模 | 参与方 | 基础持续 | 上限 | 说明 |
|----------|--------|---------|------|------|
| 局部冲突 | 2国 | 90秒 | 180秒 | 边境摩擦级别 |
| 常规战争 | 2国 | 150秒 | 300秒 | 全面交战 |
| 多国战争 | 3-4国 | 180秒 | 360秒 | 联盟战争 |
| 全面战争(L5) | 5-7国 | 120秒 | 强制结算 | 世界大战，没有赢家 |

### 4.2 战争结束条件

**1. 经济崩溃投降**

```
投降条件: military_economy降到0以下
military_economy = 初始值 - (战争消耗/tick + 贸易封锁损失 + 本国股市跌幅×杠杆)

每tick战争消耗:
├── 大国（伽蓝/铸链/暗棘）: 初始GDP的0.5%
├── 中等（焰砂/银脊/碧芒）: 初始GDP的0.8%
└── 小国（汐潮）: 初始GDP的1.2%
```

**2. 玩家促和**

| 方式 | 条件 | 效果 |
|------|------|------|
| 白鸽政策建议 | 关系>50的第三国可提出和谈 | 双方进入停火谈判（15秒） |
| 贿赂 | 花$2M-$10M | 小国可被收买停战，大国拒绝 |
| 假新闻"国际调停" | 可信度>70 | 双方暂停10秒，失效后恢复交战 |
| 经济制裁威胁 | 白鸽联合多国威胁制裁优势方 | 关系>60的≥2国参与即可 |

**3. 战争疲劳停火**

```
战争持续超过上限 → 触发停火:
├── 双方回到"冷战"关系（不是和平）
├── 双方经济都受损
├── 没有明确赢家
└── 可能30秒后再次爆发（冷战→热战概率20%）
```

**4. 一方被消灭（仅限小国）**

```
碧芒/汐潮这类小国:
├── military_economy归零 → "政权崩溃"
├── 该国货币归零，所有资产暂停交易
├── 占领国接管其资源
└── 玩家如果持有该国资产 → 全部归零
```

**5. 升级为全面战争（L5）**

```
当3个以上国家卷入 → 有概率升级为L5终局事件:
├── 所有国家被迫选边站
├── 全球市场进入"末日模式"
├── 120秒后强制结算
└── 触发结局"见证历史"或"世界终结"
```

### 4.3 胜负判定

```
战争分数 = 击杀经济分 + 资源占领分 + 外交盟友分

分数差 > 3:1 → 完胜/惨败
分数差 > 1.5:1 → 小胜/小败
分数差 < 1.5:1 → 平局
双方经济都崩溃 → 双输
```

### 4.4 战后后果

**完胜方获得：**

| 后果 | 效果 | 持续 |
|------|------|------|
| 战争赔款 | 败方GDP的10-20%转入胜方 | 一次性 |
| 资源控制 | 败方1-2个核心资源的50%产出归胜方 | 本局剩余时间 |
| 货币霸权 | 胜方货币+5%~15% | 60秒渐退 |
| 股市反弹 | 胜方军工股-20%但整体股市+10% | 脉冲 |
| 外交威望 | 所有国家对该国关系+10~20 | 本局永久 |

**惨败方承受：**

| 后果 | 效果 | 持续 |
|------|------|------|
| 战争赔款 | GDP扣10-20% | 一次性 |
| 资源丧失 | 核心资源产出-50% | 本局剩余时间 |
| 货币暴跌 | 货币-15%~30% | 60秒渐恢复到-10% |
| 股市崩跌 | 整体-15%~25% | 脉冲+缓慢恢复 |
| 社会动荡 | 国内情绪锁定在恐慌(<30) | 30秒 |
| 国际地位 | 所有关系-10~15 | 本局永久 |

**平局双方：**

| 后果 | 效果 |
|------|------|
| 双方经济受损 | GDP各-5%~10% |
| 关系降至"冷战" | 不会再开战但也无法合作 |
| 军工股回调 | 双方军工股-10%~20% |
| 粮食/石油影响持续 | 战争期间的价格变动缓慢回落 |

### 4.5 战后特殊事件

战争结束后60秒内，可能触发以下随机事件：

| 事件 | 概率 | 效果 |
|------|------|------|
| 战后重建潮 | 30% | 败方基建股暴涨20-40%，胜方建材出口股涨 |
| 战争罪审判 | 20% | 败方领导人被"审判"，该国政治动荡延续 |
| 难民危机 | 15% | 邻国经济受压，碧芒粮食需求激增 |
| 军火黑市 | 15% | 剩余军火流入暗网，暗影角色可低价购买 |
| 和平红利 | 10% | 全球情绪瞬间恢复到60+，所有风险资产反弹 |
| 复仇阴谋 | 10% | 败方暗中准备反击，60秒后可能再次开战 |

### 4.6 多场战争累积效应

```
本局内战争次数的影响:
├── 第1场: 正常
├── 第2场: 全球情绪基准-5（战争疲劳）
├── 第3场: 全球情绪基准-10，避险资产永久溢价+10%
├── 第4场+: 每多一场，L5全面战争概率+15%
└── 第4场时L5概率已达60%——几乎必然走向终局
```

### 4.7 玩家在战争中的获利路径

```
战争期间:
├── 做多胜方军工股 + 做空败方货币 → 最稳健
├── 囤积避险资产（龙头金/永恒）→ 战后抛售
├── 做多粮食/石油 → 战争推高资源价
├── 白鸽: 推动战争走向 → 政策获利
├── 暗影: 卖军火给双方 → 两头赚
├── 教父: 制造战争恐慌 → 操控情绪获利
├── 赌徒: 全押战争结果 → 极高风险极高回报
└── 先知: 预测战争走向 → 精准布局

战后:
├── 抄底败方资产 → 需要判断是V型还是L型
├── 买入重建概念股 → 碧芒基建/铸链重工
├── 做多和平红利 → 全局情绪恢复利好
└── 赌"复仇阴谋"是否会爆发 → 再次布局战争资产
```

## 五、AI KOL对战裁决机制

### 5.1 对战触发

```
触发条件（满足任一）:
├── 玩家公开质疑AI KOL → 对方必须应战（不回应则公信力-10）
├── AI KOL质疑玩家 → 玩家选择应战或无视
│   ├── 应战 → 进入对战
│   └── 无视 → 公信力-5（但不判定"输"）
├── AI KOL之间观点对立 → 自动触发（玩家旁观，可押注）
└── 玩家发"反直觉论调"且与AI KOL观点相反 → 自动触发
```

### 5.2 对战流程（30秒，6个round）

**Round 1-2: 立场交锋（各5秒）**
- 双方各发一条推文表明立场
- 市场出现分歧，部分AI跟随多方，部分跟随空方
- 双方粉丝开始"站队"

**Round 3-4: 论据对抗（各5秒）**
- 双方各抛出一个论据
- 论据质量 = 公信力×0.3 + 近期预测准确率×0.3 + 数据/新闻支撑×0.2 + 随机×0.2
- 论据更强的一方获得"气势优势"

**Round 5: 致命一击（5秒）**
- 双方各出终极论点，效果是前面几轮的2倍
- 可选特殊操作:
  - "晒单"（展示盈利，赌注$50K+）
  - "专家引用"（引用权威，成本$20K）
  - "曝光黑料"（攻击对方，风险高）
  - "以身作则"（宣布自己按说的操作，赌注最大）
  - 沉默（放弃致命一击）

**Round 6: 市场裁决（5秒）**
- 对战期间目标资产的实际价格走势决定"谁对了"
- 走势支持谁 → 谁就是最终赢家

### 5.3 裁决公式

```
对战分数 = round_score + 额外加成

round_score（每轮累积）:
├── 论据质量分: -100 ~ +100（每轮）
├── 气势优势: 连续赢round → 每轮额外+15
├── 粉丝声援: 粉丝量/对方粉丝量 × 20
└── 特殊操作加分: 晒单+30, 专家引用+20, 曝光黑料+40(成功)/-40(失败)

最终裁决:
```

### 10.1 存档时机限制

| 场景 | 能否保存 | 说明 |
|------|---------|------|
| 正常游戏中 | 随时保存 | |
| 暂停状态 | 可保存 | 最佳保存时机 |
| 战争进行中 | 可保存 | 恢复时从当前战争阶段继续 |
| KOL对战进行中 | 不可保存 | 对战仅30秒，等打完再存 |

| 结算界面 | 自动保存 | 无需手动 |
| 事件编辑器打开时 | 不可保存 | 关闭编辑器后可存 |
| 交易确认弹窗 | 不可保存 | 交易完成/取消后可存 |

### 10.2 存档内容

```
SaveData:
├── 游戏状态:
│   ├── 当前时间、速度倍率、游戏模式
│   ├── 所有资产当前价格和历史K线数据
│   ├── 三层情绪系统当前值
│   ├── 国家关系矩阵
│   └── 所有国家经济指标
├── 玩家状态:
│   ├── 资金、持仓、挂单
│   ├── 公信力、粉丝数、热度
│   ├── 天赋树进度
│   ├── 冷却计时器（所有操作的剩余冷却）
│   └── 历史记录（已发假新闻数、被揭穿数等）
├── AI状态:
│   ├── 所有AI实例的属性、持仓、资金
│   ├── AI KOL的推文历史和关系值
│   └── 骗子AI的发币状态
├── 事件状态:
│   ├── 已触发的事件列表
│   ├── 正在执行的战争（阶段、分数、消耗）
│   ├── 下次事件触发倒计时
│   └── 被激活的事件链进度
├── 随机种子:
│   └── 当前RNG种子（确保读档后随机序列一致）
└── 版本号（用于存档兼容性检查）
```

### 10.3 读档恢复规则

| 恢复场景 | 处理方式 |
|----------|---------|
| 战争进行中存档 | 读档后战争继续，阶段/分数/消耗完整恢复 |
| 有冷却中的操作 | 读档后冷却继续倒计时（不会重置） |
| 有挂单在队列中 | 读档后挂单保留，检查当前价格是否已触发 |
| 假新闻传播中 | 读档后假新闻继续传播和衰减 |
| 版本不兼容 | 提示"存档版本过旧"，安全降级不崩溃 |
| 损坏的存档 | 检测校验和，损坏时提示"存档已损坏" |

### 10.4 自动保存

```
自动保存规则:
├── 每隔5分钟自动保存一个快照（覆盖式）
├── 重大事件触发时额外保存（战争爆发、黑天鹅、崩盘）
├── 最多保留: 1个自动存档 + 1个事件存档 = 2个自动槽位
├── 手动存档槽位: 3个（玩家自由命名）
└── 沙盒模式: 只保留1个手动存档槽位
```

## 十四、赌徒运气值机制

### 14.1 运气值核心

```
运气值范围: 0-100，初始值50

自然变化:
├── 每tick随机波动 ±0.5
├── 连续盈利（3笔以上）→ 运气 -3/tick（物极必反）
└── 连续亏损（3笔以上）→ 运气 +3/tick（否极泰来）
```

### 14.2 运气值影响

| 运气区间 | 影响 |
|----------|------|
| 80-100（大运） | 命运事件80%正面，直觉准确率+20%，小概率触发"天降横财" |
| 60-79（好运） | 命运事件65%正面，直觉准确率+10% |
| 40-59（平常） | 命运事件50/50，直觉准确率正常 |
| 20-39（厄运） | 命运事件65%负面，直觉准确率-10% |
| 0-19（大凶） | 命运事件80%负面，直觉准确率-20%，小概率触发"祸不单行" |

### 14.3 运气值变化来源

```
增加运气:
├── 全押成功 → +15
├── 逆风翻盘成功 → +30
├── 亏损后继续交易 → +2/笔（赌徒不认输）
├── 命运轮盘（消耗20运气赌更大） → 成功+40
└── 天赋"运气操控"：花$50K → +20

减少运气:
├── 全押失败 → -25
├── 爆仓 → -30
├── 赚大钱后继续豪赌 → -5/笔（贪心惩罚）
└── 命运轮盘失败 → -20（额外扣除）
```

### 14.4 命运事件

```
每60-120秒触发一次命运事件（受天赋影响频率）

命运事件类型:
├── 天降横财: 随机获得$1K-$50K
├── 内幕泄露: 获知一条即将发生的事件
├── 赌神附体: 下一笔交易收益×2
├── 幸运捡漏: 发现一个被低估的资产
├── 黑客入侵: 账户被黑，损失5-15%资金
├── 噩耗传来: 持仓最重的资产暴跌5-10%
├── 被盯上了: 随机一个国家对你发起调查
├── 赌场风云: 被迫参加一次命运轮盘（全押或放弃一半资金）
└── 平淡无奇: 什么都没发生（最无聊的命运）
```

## 十八、玩家发行币价格公式

```
价格 = 流动性池资金 / (总供应量 × 流通比例)

其中:
├── 流通比例 = 1 - 预留筹码比例
├── 买入时: 价格上升，上升幅度 = 买入金额 / 流动性池 × 当前价格
├── 卖出时: 价格下降，下降幅度 = 卖出金额 / 流动性池 × 当前价格
├── 预留筹码不可交易（直到Rug时释放）
└── Rug瞬间: 流动性池归零，价格 = 0，所有持仓者归零

特殊规则:
├── 无杠杆、无做空
├── 无熔断（涨跌不限制）
├── 交易手续费1.0%
├── 仅在汐潮交易所交易
└── 不计入崩盘指数（法外资产）
```
