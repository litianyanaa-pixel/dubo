# 《运筹帷幄》游戏设计文档（讨论记录）

---

## 项目定位

- 游戏类型：单机 / Roguelike / 黑色幽默 / 全球金融市场操控模拟器
- 核心体验：玩家通过操控舆论、散播假新闻、制造 FOMO 与恐慌，在一个模拟全球金融市场中不断获利，并在崩盘中活下来
- 游戏真正模拟的：不是炒币，而是互联网时代的群体情绪——人性、贪婪、恐惧、FOMO、舆论、信仰
- 参考：《Balatro》《Luck be a Landlord》《Papers, Please》这种"规则系统驱动爽感"的游戏
- 美术方向：低成本高风格化，UI驱动型游戏，避免3D大场景
- UI风格：赛博金融终端，融合 TradingView + Bloomberg Terminal + 暗网UI + Meme Culture
- 技术栈：React + TypeScript + TailwindCSS + Framer Motion + Zustand + Lightweight Charts + Howler.js + Tauri

## 一、国家系统（7个虚构国家）

### 设计原则
- 不映射真实国家，避免政治麻烦
- 每个国家至少有一个"只可能存在于游戏中"的独特设定
- 每个国家有深度模拟：经济基本面、政策系统、国际关系、国内局势、市场状态

### 1.1 伽蓝联邦（Kalan Federation）

> "算法治国，效率至上，感情是多余的变量。"

**起源**：三座科技城邦合并而成。最早的创始人是一群相信"人类不该治理人类"的工程师。

**经济人格：算法霸权**

| 维度 | 内容 |
|------|------|
| 核心产业 | AI、量子计算、芯片、自动化、生物科技 |
| 货币 | 伽蓝币（KAL）——全球结算首选，信用最强 |
| 独特设定 | 国家政策由AI"天算"辅助决策，每72秒自动调整利率、税收和产业补贴。市场参与者永远在猜"天算"下一步会怎么算 |
| 军事 | 无人化军队，AI指挥，数量少但精准 |
| 国内矛盾 | 反对"机器治国"的地下运动、AI与人类就业冲突、技术寡头垄断 |

**市场特征**：
- "天算"调整政策的瞬间，市场会剧烈波动——这是玩家可以利用的节奏
- 科技突破会带动全球科技板块

**政策倾向**：
- 加息 → 全球资本回流，其他货币贬值
- 制裁 → 目标国货币暴跌、产业瘫痪
- 技术突破 → 自身股市暴涨，相关全球板块跟涨

### 1.2 焰砂汗国（Ember Khanate）

> "地底埋着整个世界的欲望。"

**起源**：千年游牧帝国定居在一片资源极其丰富的荒漠之上。汗室控制所有矿脉，用资源换取现代化的武器和基建。

**经济人格：资源独裁**

| 维度 | 内容 |
|------|------|
| 核心产业 | 石油、稀土、地热能源、矿产 |
| 货币 | 焰砂元（EMK）——绑定资源出口价格，涨跌看天 |
| 独特设定 | 每年举行一次"资源拍卖会"，各国竞标开采权，拍卖结果直接影响全球大宗商品价格 |
| 军事 | 雇佣军体系+资源外交，控制两条全球关键航道 |
| 国内矛盾 | 汗室诸王子夺嫡、游牧传统vs现代化、被剥削的矿工阶层 |

**市场特征**：
- 资源拍卖会是全年最大行情事件
- 一旦卷入继承危机，市场不确定性飙升

**政策倾向**：
- 减产 → 全球油价飙升，制造业国家成本暴涨
- 断供 → 目标国经济受冲击，能源股崩盘
- 扩产 → 油价暴跌，新能源产业受打击

---

### 1.3 铸链联合体（Forge-Chain Commonwealth）

> "每个人都是链条上的一环，断裂者将被替换。"

**起源**：工业革命从未结束的国家。整片国土就是一座工厂，公民即工人、工人即股东。

**经济人格：超级工厂**

| 维度 | 内容 |
|------|------|
| 核心产业 | 制造业、基建、新能源设备、代工 |
| 货币 | 铸链币（FOR）——稳定但有管制，存在黑市汇率 |
| 独特设定 | "全民持股制"——每个公民持有国家工业的股份，民众情绪直接影响股票指数。民众不满→抛售→股灾→社会动荡的恶性循环 |
| 军事 | 大规模、工业化战争能力，以量取胜 |
| 国内矛盾 | 产能过剩、环境崩溃、年轻一代拒绝"出厂" |

**市场特征**：
- 股市和民众情绪深度绑定，舆论操控可以引发股灾
- 制造业数据直接影响全球供应链

**政策倾向**：
- 降准降息 → 资本外流压力，国内股市刺激
- 资本管制升级 → 黑市汇率飙升，外资恐慌
- 产业补贴 → 特定板块暴涨（新能源、芯片）

---

### 1.4 银脊共和国（Silver Ridge Republic）

> "你的秘密很安全，只要你能付得起保管费。"

**起源**：群山之中的堡垒城邦。历史上从未被攻破，逐渐成为全球资金的避难所。

**经济人格：金融堡垒**

| 维度 | 内容 |
|------|------|
| 核心产业 | 银行业、保险、信托、离岸金融、精密仪器 |
| 货币 | 银脊盾（SRD）——全球最稳定货币，避险首选之一 |
| 独特设定 | "绝对保密法"——银脊的银行不向任何政府披露客户信息。但每隔一段时间，银脊会秘密拍卖"情报包"——各国权贵在这里的资产明细。买到的玩家可以获得巨大的政治筹码 |
| 军事 | 地形天然要塞，极小规模但坚不可摧 |
| 国内矛盾 | 银行业绑架政治、洗钱争议、贫富悬殊到荒谬 |

**市场特征**：
- 银脊的利率变化是全球资本流向的风向标
- "情报包"拍卖是高风险高回报的玩法

**政策倾向**：
- 降息 → 全球热钱涌入风险资产
- 加息 → 资金回流，全球流动性收紧
- 突然取消保密政策 → 全球恐慌，暴露无数黑账户

---

### 1.5 暗棘帝国（Dark Thorn Empire）

> "我们不需要繁荣，我们只需要别人比我们更弱。"

**起源**：军国主义帝国的残骸。曾经辉煌，如今封闭、偏执、用恐惧维持统治。

**经济人格：铁幕孤岛**

| 维度 | 内容 |
|------|------|
| 核心产业 | 军工、重工业、网络战、核能 |
| 货币 | 暗棘令（DTR）——不可自由兑换，官方汇率与黑市汇率差3-10倍 |
| 独特设定 | "影子市场"——暗棘内部存在完全独立的黑市经济，黑市汇率才是真实汇率。玩家可以通过特殊渠道进入黑市套利，但被抓概率随操作频率递增 |
| 军事 | 庞大但老旧、核威慑、网络战能力世界顶尖 |
| 国内矛盾 | 物资短缺、寡头贪腐、年轻一代偷渡外逃 |

**市场特征**：
- 官方/黑市双汇率是天然套利空间
- 每次军事挑衅都会引发全球市场震荡

**政策倾向**：
- 军事演习 → 周边国家恐慌，军工股涨，旅游跌
- 入侵邻国 → 全球市场剧震，油价飙升，资本涌入避险资产
- 突然开放 → 外资涌入抄底，短期暴涨后可能被收割

### 1.6 汐潮自由邦（Tidal Free State）

> "我们的国界随潮汐移动，法律只在你踩得到的地方有效。"

**起源**：由海洋平台、废弃油轮、人工岛屿拼接而成的漂浮国度。不属于任何大陆，却连接所有航线。

**经济人格：流动天堂**

| 维度 | 内容 |
|------|------|
| 核心产业 | 虚拟货币交易所、离岸注册、航运、灰色金融、自由博彩 |
| 货币 | 汐潮元（TDL）——与一篮子货币+主流虚拟币挂钩 |
| 独特设定 | "法外经济区"——汐潮没有金融监管，任何在其他国家违法的操作在这里合法。玩家可以在这里洗钱、发行空气币、操纵市场，不受任何追诉。但汐潮本身也最不稳定——可能被飓风摧毁、被海盗劫持、被多国联合施压 |
| 军事 | 无正规军，靠雇佣和地理优势（无法被陆军攻击） |
| 国内矛盾 | 海盗渗透、环境污染、基础设施脆弱 |

**市场特征**：
- 玩家发行虚拟币的唯一合法通道
- 汐潮被施压或遭灾时，虚拟币市场会剧烈波动

**政策倾向**：
- 监管放松 → 全球灰色资本涌入，虚拟货币暴涨
- 突然配合制裁 → 某个国家资本被冻结，市场混乱
- 新自由港政策 → 资金从银脊流出，金融中心地位动摇

### 1.7 碧芒同盟（Jade Light Alliance）

> "别人造泡沫，我们种粮食。泡沫会破，粮食不会。"

**起源**：一群农业大国和新能源先驱组成的松散联盟。不追求霸权，但掌控着所有人都要吃饭的事实。

**经济人格：绿色粮仓**

| 维度 | 内容 |
|------|------|
| 核心产业 | 粮食、生物科技、新能源、稀有作物、碳交易 |
| 货币 | 碧芒币（JDL）——低波动，跟随粮食周期 |
| 独特设定 | "全球种子库"——碧芒掌握着全球70%的粮食品种专利。任何国家想种粮食，都要向碧芒交专利费。这不是武器，但比武器更有效 |
| 军事 | 防御为主，但拥有全球最先进的生物防御技术 |
| 国内矛盾 | 联盟内部各国利益不统一、环保与发展的拉锯 |

**市场特征**：
- 粮食禁运是最致命的非军事武器
- 新能源技术突破会直接冲击焰砂和铸链

**政策倾向**：
- 粮食减产 → 全球粮价飙升，进口国通胀加剧
- 新能源技术突破 → 石油需求预期下降，焰砂受冲击
- 出口禁令 → 依赖国恐慌，粮食期货暴涨

### 1.8 国家属性结构

每个国家需要深度模拟的维度：

```
国家属性:
├── 经济基本面
│   ├── GDP增速
│   ├── 通胀率
│   ├── 利率
│   ├── 失业率
│   └── 财政赤字
├── 政策系统
│   ├── 货币政策（加息/降息/QE）
│   ├── 贸易政策（关税/制裁/自贸协定）
│   ├── 产业政策（补贴/限制/国有化）
│   └── 资本管制（开放/限制/冻结）
├── 国际关系
│   ├── 与每个其他国家的关系值（-100 ~ +100）
│   ├── 贸易依赖度
│   └── 军事/政治紧张度
├── 国内局势
│   ├── 民众满意度
│   ├── 政府稳定性
│   └── 黑天鹅 susceptibility
└── 市场状态
    ├── 股票指数
    ├── 主权货币汇率
    ├── 国债收益率
    └── 资本流入/流出
```

### 1.9 国家间冲突系统

#### 冲突光谱（五级）

```
层级0: 和平期
  → 自贸、技术合作、文化交流
  → 市场：稳定增长，波动低

层级1: 摩擦期
  → 贸易战、关税、外交抗议
  → 市场：相关板块波动加大

层级2: 冷战期
  → 制裁、技术封锁、间谍战、代理人冲突
  → 市场：资本撤离、避险资产涨

层级3: 热战期
  → 军事冲突、封锁、有限战争
  → 市场：全面恐慌、军工涨、一切跌

层级4: 全面战争
  → 大规模战争、核威胁、全球经济崩溃
  → 市场：归零级崩盘（可能触发游戏一局结束）
```

#### 冲突形态详解

```
和平 ←──────→ 冷战 ←──────→ 局部冲突 ←──────→ 全面战争
  │                │               │                  │
 自贸协定        制裁/禁运      代理人战争        入侵/占领
 技术共享        情报战         军事演习          核威胁
 联合军演        贸易壁垒       海上封锁          全面制裁
 文化交流        间谍活动       网络攻击          资源掠夺
```

#### 战争对市场的影响

- 铁翼入侵邻国 → 全球股市暴跌、油价暴涨、避险资产飙升、军工股暴涨
- 伽蓝制裁铸链 → 芯片断供、铸链科技股崩盘、伽蓝芯片股涨、替代供应链暴涨
- 焰砂减产 → 制造业成本飙升、铸链出口受打击、碧芒新能源受益
- 汐潮被施压关闭交易所 → 虚拟货币市场恐慌、资金流向黑市

#### 战争触发

战争可以由**AI国家行为、黑天鹅事件、玩家暗中挑拨**触发。玩家也可以通过操控舆论、情报来阻止或加速战争。

## 二、角色系统（6个角色）

### 设计原则
- 每个角色有：背景、初始条件、主动技能、被动特性、独有机制、弱点、成长路线
- 不同角色 = 不同玩法风格，不是简单的数值差异

### 2.1 华尔街猎犬·陈

> "我不赌方向，我只赌你们的恐惧。"

**背景**：前顶级对冲基金经理，因一次做空被华尔街封杀。现在以个人身份回到市场，带着他最可怕的武器——对人性的精准判断。

**初始条件**：
- 资金：★★★★★（极高）
- 影响力：★★☆☆☆（低，被主流封杀）
- 关系网：金融圈内线人
- 起始地区：银脊共和国

**主动技能**：

| 技能 | 效果 | 冷却 |
|------|------|------|
| 内幕消息 | 获取一条即将发生的政策/事件情报 | 60秒 |
| 精准做空 | 做空时手续费减半，杠杆上限+50% | 被动 |
| 机构联动 | 调动关联资金跟风操作，放大市场影响 | 90秒 |
| 紧急平仓 | 免手续费瞬间全部平仓（避免爆仓） | 180秒 |

**被动特性**：
- 「冷血」：市场恐慌时，自身操作不会被情绪干扰，分析准确率+30%
- 「旧敌」：每局随机一个AI机构会主动针对你做空

**独有机制——信息差交易**：
- 可以花大价钱购买未公开的政策情报、企业财报、央行决议
- 在信息公开前布局，信息公开后获利

**弱点**：
- 舆论操控能力极弱，发假新闻效果只有其他角色的30%
- 粉丝增长极慢

**成长路线**（通过天赋树系统，详见数值文档）：
```
情报网络树: 内幕消息强化 → 央行政策预判 → 多渠道情报 → 央行间谍
金融操盘树: 手续费减免 → 杠杆增强 → 大单拆分 → 做市商模式
市场洞察树: 技术指标增强 → 波动率预测 → 连锁预判 → 崩盘预警
生存能力树: 爆仓保护 → 法律风险降低 → 紧急平仓 → 金融堡垒
```

### 2.2 舆论教父·Z

> "我说的每一句话，明天就会变成K线。"

**背景**：社交媒体上拥有千万粉丝的神秘KOL，没人知道真实身份。一条推文就能让一个币归零或者起飞。

**初始条件**：
- 资金：★★☆☆☆（中低）
- 影响力：★★★★★（极高）
- 关系网：各国KOL、媒体人脉
- 起始地区：伽蓝联邦

**主动技能**：

| 技能 | 效果 | 冷却 |
|------|------|------|
| 喊单推文 | 发一条针对特定资产的推文，粉丝跟风买卖 | 30秒 |
| 制造话题 | 发起一个热门话题标签，影响板块情绪 | 45秒 |
| 直播喊话 | 开启直播，持续影响市场情绪（持续15秒） | 120秒 |
| 转发炸弹 | 让其他KOL转发你的观点，影响力×3 | 90秒 |

**被动特性**：
- 「粉丝经济」：粉丝数直接影响市场操控范围，粉丝越多效果越强
- 「信用崩塌」：每次预测失败/被抓到撒谎，粉丝流失10-30%

**独有机制——粉丝帝国**：
- 粉丝数量是核心资源，粉丝越多，一句话对市场的影响越大
- 粉丝会自发交易（AI模拟），形成真实的"KOL效应"
- 可以与其他KOL结盟或对抗

**弱点**：
- 资金量小，自己吃不到自己制造的行情的全部利润
- 被打假一次，公信力暴跌

**成长路线**（通过天赋树系统，详见数值文档）：
```
舆论操控树: 传播增强 → 可信度提升 → 多平台发布 → 造神运动
粉丝经济树: 铁粉占比提升 → 增长加速 → 付费社群 → KOL军团
商业变现树: 广告收入增加 → 收费喊单 → 暗盘交易 → 媒体帝国
生存能力树: 公信力保护 → 恢复加速 → 危机公关 → 金蝉脱壳
```

| **L4 宏观层** | 每30秒 | 国家经济指标更新、政策评估、国际关系变化 |

### 4.3 价格引擎

#### 核心公式

每个资产每个L0 tick，价格变化量：

```
ΔPrice = Price × (
    f_fundamental   // 基本面影响
  + f_sentiment     // 情绪影响
  + f_ai_flow       // AI资金流
  + f_player        // 玩家操作
  + f_correlation   // 跨市场联动
  + f_noise         // 随机噪声
)
```

每一项都是 -1 ~ +1 之间的小数，乘以各自的权重系数后求和。

---

#### 4.3.1 基本面影响 f_fundamental

来自国家和资产的经济基础数据：

```
f_fundamental = w1×GDP增速 + w2×利率差 + w3×通胀偏差
              + w4×贸易余额 + w5×政策变动 + w6×产业数据
```

举例：
- 伽蓝加息0.5% → KAL汇率基本面因子 +0.3%（资本流入）
- 铸链PMI跌破50 → FOR基本面因子 -0.5%（制造业萎缩）
- 焰砂发现新矿脉 → 相关大宗商品基本面 +0.8%

权重 w1-w6 按资产类型不同：
- 货币/汇率：利率差权重最高
- 股票：GDP和产业数据权重最高
- 大宗商品：供需和库存权重最高
- 龙头金/永恒：几乎不受基本面影响（纯情绪驱动）

---

#### 4.3.2 情绪影响 f_sentiment

情绪-价格映射函数：

```
情绪值范围: 0 ─── 25 ─── 50 ─── 75 ─── 100
情绪状态:  极度恐慌  恐慌   中性   贪婪   极度贪婪

价格影响:   -0.8%   -0.3%   0%   +0.3%   +0.8%
                     ↑                   ↑
                  线性区              非线性加速区
```

关键设计：**极端区域是非线性的**

- 情绪在30-70之间时，价格影响是线性的、温和的
- 情绪超过80或低于20时，进入**非线性加速区**——价格开始加速涨/跌
- 超过90或低于10时，进入**疯狂区**——可能出现熔断、闪崩、暴涨

不同资产对情绪的敏感度：

| 资产类型 | 情绪敏感度 | 说明 |
|----------|------------|------|
| 虚拟币/Meme币 | 极高 | 几乎全靠情绪驱动 |
| 新兴市场股票 | 高 | 容易恐慌 |
| 大宗商品 | 中 | 受供需影响更多 |
| 主权货币 | 低 | 相对稳定 |
| 龙头金 | 负相关 | 市场越恐慌，龙头金越涨 |
| 永恒 | 极低 | 几乎免疫情绪 |

---

#### 4.3.3 AI资金流 f_ai_flow

所有AI这个tick的买卖总和：

```
f_ai_flow = Σ(AI买入量 - AI卖出量) / 当前流动性
```

---

#### 4.3.4 玩家操作 f_player

```
f_player = (玩家买入量 - 玩家卖出量) / 当前流动性
```

关键参数：**流动性**

- 流动性高的资产：玩家操作影响小（如KAL/USD）
- 流动性低的资产：玩家操作影响大（如小国货币、Meme币）
- 这意味着：玩家用少量资金就能操纵流动性低的市场

---

#### 4.3.5 跨市场联动 f_correlation

通过关联矩阵计算：

```
关联矩阵示例（简化）:

          KAL   EMK   FOR   SRD   DTR   石油   龙头金
KAL       1.0  -0.3  -0.2   0.1  -0.5  -0.1    0.2
EMK      -0.3   1.0   0.2  -0.1   0.1   0.9   -0.3
FOR      -0.2   0.2   1.0   0.0   0.2  -0.2    0.1
龙头金    0.2  -0.3   0.1   0.3  -0.2  -0.3    1.0
```

联动计算：
```
f_correlation = Σ(关联资产价格变化 × 关联系数)
```

举例：
- 石油涨了1% → EMK联动 +0.9%（焰砂货币跟着涨）
- 石油涨了1% → 龙头金联动 -0.3%（避险需求降低）

**板块联动**：同一板块的资产有额外联动加成
- 科技板块：伽蓝科技股 + 铸链新能源股
- 资源板块：石油 + 天然气 + 矿产
- 军工板块：暗棘军工 + 伽蓝无人机

---

#### 4.3.6 随机噪声 f_noise

```
f_noise = 基础波动率 × N(0, 1) × 波动率乘数
```

| 资产类型 | 基础波动率 |
|----------|------------|
| 主权货币 | 0.1-0.3% |
| 蓝筹股 | 0.2-0.5% |
| 大宗商品 | 0.3-0.6% |
| 新兴市场 | 0.5-1.0% |
| 虚拟币 | 1.0-3.0% |
| 玩家发行币 | 2.0-5.0% |

---

#### 4.3.7 价格保护机制

```
单tick最大涨跌幅:
  正常状态: ±3%
  情绪极端(>90或<10): ±8%
  黑天鹅事件: ±15%
  熔断: 触达阈值后暂停交易10秒（冷却）
```

## 五、情绪引擎

### 5.1 三层结构

```
┌─────────────────────────────────┐
│  全局情绪 GlobalSentiment        │  ← 整个市场的总基调
│  范围: 0-100                     │
├─────────────────────────────────┤
│  国家情绪 CountrySentiment       │  ← 每个国家的市场信心
│  范围: 0-100 × 7个国家           │
├─────────────────────────────────┤
│  资产情绪 AssetSentiment         │  ← 单个资产的交易者情绪
│  范围: 0-100 × N个资产           │
└─────────────────────────────────┘
```

影响关系：
```
全局情绪 → 影响所有国家情绪（权重高）
国家情绪 → 影响该国资产情绪（权重高）
资产情绪 → 反向影响全局情绪（权重低，但暴涨暴跌时权重陡增）
```

---

### 5.2 情绪更新公式（每L2 tick）

**全局情绪**：
```
GlobalSentiment(t) = GlobalSentiment(t-1)
  + Σ(国家情绪变化 × 该国经济权重)
  + 黑天鹅冲击
  + 惯性衰减（向50回归，速度0.5/tick）
```

**国家情绪**：
```
CountrySentiment(t) = CountrySentiment(t-1)
  + 全局情绪传导（差值 × 0.1）
  + 国内政策影响
  + 战争/冲突影响
  + 资本流入/流出信号
  + 惯性衰减（向50回归，速度0.3/tick）
```

**资产情绪**：
```
AssetSentiment(t) = AssetSentiment(t-1)
  + 国家情绪传导
  + 新闻/假新闻冲击
  + KOL喊单影响
  + 价格走势反馈（涨→情绪升，跌→情绪降）
  + AI集体行为反馈
  + 惯性衰减（向50回归，速度1.0/tick）
```

---

### 5.3 情绪突变机制

大多数时候情绪是渐变的（惯性衰减+逐步传导），但某些触发条件会导致**瞬间跳变**：

| 触发事件 | 跳变幅度 | 恢复方式 |
|----------|----------|----------|
| 战争爆发 | -30~-50 | 缓慢恢复，战争结束前不会恢复到中性 |
| 央行紧急加息 | -15~-25 | 3-5个L2 tick后开始消化 |
| 重大科技突破 | +15~+25 | 衰减较快 |
| 玩家制造爆炸性假新闻 | -20~+20（看内容） | 取决于是否被揭穿 |
| 大型Rug Pull | -25~-40 | 长期信心受损 |
| 龙头金暴涨（避险信号） | -10~-15 | 正反馈：龙头金涨→恐慌加剧→龙头金再涨 |
| KOL百万粉喊单 | +10~+20 | 快速衰减，1-2个tick |

**揭穿机制**：假新闻如果被揭穿（AI自动检测或对手KOL拆穿），情绪会**反向跳变**——之前涨多少，现在加倍跌回来。

---

### 5.4 情绪的"记忆"效应

情绪系统不是无状态的。它有记忆：

```
情绪惯性 = 历史情绪加权平均（越近的权重越高）

影响：
- 连续上涨后的下跌 → 恐慌更剧烈（"到手的钱没了"）
- 连续下跌后的上涨 → 反弹更猛烈（"终于回本了"）
- 长期中性 → 波动缩小（市场"睡着了"）
```

## 六、AI决策引擎

### 6.1 AI个体属性

每个AI个体都有独立的属性：

```
AI个体:
├── 类型（韭菜/鲸鱼/KOL/骗子/国家机构）
├── 资金量
├── 风险偏好（0-100）
├── 情绪敏感度（0-100）
├── 理性程度（0-100）
├── 持仓（各资产的持仓量）
├── 恐惧阈值（低于此情绪值会恐慌卖出）
├── 贪婪阈值（高于此情绪值会FOMO买入）
├── KOL信任列表（跟随哪些KOL）
└── 信息来源偏好（看新闻/看数据/听KOL/看K线）
```

同一类型内部的AI也有差异——不是所有韭菜都一样。

---

### 6.2 AI决策流程（每L1 tick）

```
┌─────────────────────────────┐
│  Step 1: 感知环境            │
│  读取: 价格变化、新闻、情绪、  │
│       KOL动态、自己的持仓状态  │
└──────────┬──────────────────┘
           ▼
┌─────────────────────────────┐
│  Step 2: 评估压力            │
│  计算恐惧值和贪婪值           │
└──────────┬──────────────────┘
           ▼
┌─────────────────────────────┐
│  Step 3: 优先级判定          │
│  多个信号冲突时，按优先级选    │
└──────────┬──────────────────┘
           ▼
┌─────────────────────────────┐
│  Step 4: 执行交易            │
│  决定买/卖/持有 + 仓位大小    │
└─────────────────────────────┘
```

---

### 6.3 韭菜AI

数量：200-300个实例，容量小但总量大

```
优先级（从高到低）:

1. 止损求生
   if 持仓亏损 > 风险偏好阈值:
     → 恐慌市价卖出全部持仓
     → 情绪敏感度高的韭菜更早触发

2. FOMO追涨
   if 某资产近5个tick涨幅 > 3% AND 情绪 > 70:
     → 市价买入（仓位 = 贪婪程度 × 可用资金）
     → 信号来源：K线、社交媒体热度

3. KOL跟随
   if 信任的KOL发出买入/卖出信号:
     → 延迟2-4个tick后跟随（韭菜反应慢）
     → 跟随仓位 = KOL影响力权重 × 可用资金

4. 新闻反应
   if 看到利好/利空新闻:
     → 根据新闻情绪值和可信度决定是否行动
     → 理性度低的韭菜更容易被假新闻骗

5. 羊群效应
   if 观察到大量AI在买入/卖出某资产:
     → 跟风操作（"大家都在买，我也买"）
     → 这是最弱的信号，但在极端行情中权重会放大

6. 默认
   → 小额随机交易（维持市场活力）
```

**韭菜的特征**：
- 容量小但数量多
- 亏损时恐慌极快，盈利时舍不得卖
- 容易被假新闻骗，但也容易被"打脸"后不再相信同一个来源
- **有学习能力的弱AI**：被骗过3次的韭菜会提高对同一来源的不信任度

---

### 6.4 鲸鱼AI

数量：6-10个，每个资金量巨大

```
优先级:

1. 主动操盘
   if 判断某资产被高估/低估:
     → 大规模建仓或清仓
     → 鲸鱼有独立判断能力，不完全跟情绪走

2. 收割时机
   if 韭菜大量持仓某资产 AND 鲸鱼有大量空头:
     → 砸盘（大量卖出触发韭菜恐慌）
     → 在韭菜恐慌抛售时低价回补

3. 对冲操作
   if 持仓风险过高:
     → 在关联资产上做反向对冲
     → 鲸鱼比韭菜理性得多

4. 事件交易
   if 预判到即将发生的政策/事件:
     → 提前布局
     → 鲸鱼有信息优势，能更早感知事件

5. 流动性提供
   → 在价差大时挂限价单，赚取买卖价差
```

**鲸鱼的特征**：
- 数量少但每个资金量巨大
- 单笔交易就能推动价格
- 有"狩猎"行为——会刻意制造波动吸引韭菜入场
- 互相博弈——鲸鱼之间会互相狙杀

---

### 6.5 KOL AI

```
行为模式:

1. 发布观点
   - 基于自己的持仓方向喊单（有利益动机）
   - 喊多 → 自己先建好仓位
   - 粉丝跟随 → 推高价格 → 自己高位出货

2. 转发新闻
   - 选择性地转发利好/利空新闻（配合自己持仓）
   - 给新闻加上个人解读，放大影响力

3. 互相攻讦
   - KOL之间会互相拆台
   - "他说要涨？我觉得他要割韭菜"
   - 这会制造舆论对立，增加市场波动

4. 节奏控制
   - 聪明的KOL不会一直喊单
   - 会"预测正确"几次建立信任，然后利用信任收割
```

**KOL的特征**：
- 粉丝数决定影响力
- 公信力是核心资源，被拆穿一次就大幅下降
- 有些KOL是"正直的"（真的在分析），有些纯粹是骗子
- **玩家作为KOL**可以与AI KOL竞争/合作/对抗

---

### 6.6 骗子AI

数量：每局2-3个

```
行为模式:

1. 发空气币
   - 编造一个项目叙事
   - 通过汐潮自由邦的交易所发行
   - 用AI水军制造热度

2. 拉盘吸引
   - 初期拉高价格，吸引韭菜入场

3. Rug Pull
   - 在市值最高点，突然抽走流动性
   - 所有持有者瞬间归零

4. 换皮重来
   - Rug之后换一个名字和叙事
   - 再次执行相同骗局
```

**骗子AI的特征**：
- 发行的币有"Rug Risk"属性，玩家可以通过数据分析检测到
- 玩家也可以选择"跟骗"——在Rug之前卖出获利

---

### 6.7 国家机构AI

```
行为模式:

1. 央行操作
   - 根据经济数据调整利率
   - 必要时直接干预汇率（买卖本币）
   - 伽蓝的"天算"每72秒自动评估一次政策

2. 主权基金交易
   - 大规模跨市场操作
   - 可能配合国家战略（做空敌对国货币）

3. 监管行动
   - 调查可疑交易（包括玩家的）
   - 暂停异常交易
   - 发布政策影响市场
```

---

### 6.8 AI之间的互动网络（收割链）

```
  骗子发币 → KOL喊单 → 韭菜FOMO买入
                ↑              ↓
           鲸鱼趁机拉盘   韭菜被套
                ↓              ↓
           鲸鱼高位出货   韭菜恐慌抛售
                ↓              ↓
           价格暴跌 ←─── 全市场情绪下降
                ↓
           骗子Rug Pull → 剩余韭菜归零
```

玩家可以参与其中任何一个环节——可以当骗子、当KOL、当鲸鱼，或者提前识别骗局反收割。

## 七、跨市场联动系统

### 7.1 联动触发规则

```
规则1: 替代效应
  伽蓝芯片股涨 → 铸链科技股跌（竞争关系）
  焰砂石油涨 → 碧芒新能源涨（替代逻辑）

规则2: 供应链传导
  焰砂减产 → 石油涨 → 铸链制造业成本上升 → 铸链股票跌
  碧芒粮食减产 → 粮价涨 → 所有国家通胀上升 → 央行加息 → 股市跌

规则3: 避险切换
  风险资产跌 → 龙头金涨 → 永恒涨
  恐慌消退 → 龙头金跌 → 风险资产涨

规则4: 货币三角
  KAL涨 → 其他货币相对贬值 → 新兴市场资本外流 → FOR大跌
  加息 → 资本回流加息国 → 全球流动性收紧

规则5: 板块共振
  同板块资产有0.3-0.7的关联系数
  一个涨，其他跟涨（但幅度打折）
```

---

### 7.2 传导延迟

联动不是瞬间的——有延迟，这给了玩家套利空间：

```
核心市场（KAL、石油）: 0 tick延迟
关联市场（FOR、大宗商品）: 1-2 tick延迟（0.5-1秒）
边缘市场（DTR、小国货币）: 3-5 tick延迟（1.5-2.5秒）
虚拟币: 2-4 tick延迟
```

延迟意味着：看到石油涨了，可以抢先买入EMK——但如果所有人都这么做，价格就瞬间修正了。

假新闻是玩家的**核心武器**。不只是"发一条消息"那么简单——它是一套完整的"舆论战"系统：制造→投放→传播→发酵→收割（或被揭穿反噬）。

### 8.6 传播阶段

新闻投放后不是所有人立刻看到。传播是一个病毒式扩散过程：

```
传播公式:

每tick新增曝光量 = 当前曝光量 × 传播系数 × (1 - 市场饱和度)

传播系数 = 基础传播率
         × 情绪烈度加成（越刺激传播越快）
         × 水军加成
         × KOL转发加成
         × 平台算法加成（争议性内容被推荐）

市场饱和度 = 已知悉该新闻的AI占比（0~1）
```

传播曲线：

```
曝光量
  │          ╱‾‾‾‾‾‾╲
  │        ╱         ╲
  │      ╱            ╲
  │    ╱               ╲
  │  ╱                  ╲______
  │╱                           ╲___
  └──────────────────────────────── 时间
  投放  加速传播  峰值  衰退期
```

#### 影响传播的因素

| 因素 | 影响 |
|------|------|
| 新闻的情绪冲击力 | 越刺激传播越快 |
| 可信度 | 越高传播越远（更多人愿意转发） |
| 水军助推 | 可以人为加速传播初期的启动速度 |
| KOL转发 | KOL粉丝群瞬间爆发式曝光 |
| 已有市场关注度 | 目标资产越热门，新闻传播越快 |
| 反驳信息 | 如果有人开始辟谣，传播速度减半 |

### 8.7 发酵阶段

新闻被足够多的人看到后，开始影响市场：

```
市场影响 = 新闻力度 × 可信度 × 当前曝光比例 × 情绪放大系数

情绪放大系数:
  if 市场情绪已经在极端区域(>80或<20): 放大系数 = 1.5×（火上浇油）
  else: 放大系数 = 1.0×

影响随时间衰减:
  实际影响 = 初始影响 × e^(-衰减率 × 时间)

总持续时间 = 基础时长（10-30个L2 tick）× 力度
```

### 8.8 揭穿机制

新闻不是一定能骗到底的。系统中存在自动揭穿和玩家对抗机制。

#### 揭穿概率计算

```
每tick揭穿概率 = 基础揭穿率 × 以下修正:

修正因子:
├── 新闻可信度越低 → 揭穿概率越高
├── 新闻力度越大 → 揭穿概率越高（大新闻更容易被查证）
├── 影响的资产流动性越高 → 揭穿概率越高（大市场有更多分析师）
├── 已有AI KOL发布质疑 → 揭穿概率+50%
├── 暗影操盘手的"不留痕迹"技能 → 揭穿概率-50%
├── 同一来源之前被揭穿过 → 揭穿概率+30%
└── 玩家角色的舆论操控能力 → 降低揭穿概率
```

#### 揭穿的后果

```
1. 情绪反向跳变: 之前因为假新闻产生的情绪变化全部反转，且×1.5倍惩罚
   例：假新闻让情绪+20 → 揭穿后情绪-30

2. 信用惩罚:
   ├── 玩家公信力 -15~-25（看严重程度）
   ├── 粉丝流失 -10%~-30%（教父角色额外-10%）
   └── 后续新闻的可信度永久降低10%

3. 法律风险:
   ├── 在暗棘帝国: 可能被"请喝茶"（资金冻结）
   ├── 在伽蓝联邦: 可能被AI监管系统标记（后续操作被监控）
   └── 在汐潮自由邦: 无后果（法外之地）
```

#### 玩家可以主动辟谣

- 发布反驳信息，消耗自己的公信力
- 如果辟谣成功（新闻确实是假的），自己公信力上升
- 如果辟谣失败（新闻是真的），自己公信力暴跌（"造谣者"反噬）

           OR 碧芒对3个以上国家实施出口禁令
  概率: 基础2%/分钟，前置条件满足后提升到25%/分钟
```

### 9.3 玩家触发事件

玩家不只是被动接受事件，可以主动制造或催化事件：

| 途径 | 方式 | 成本/风险 |
|------|------|-----------|
| 假新闻催化 | 持续制造关于某个国家内部矛盾的假新闻，提升该国的"不稳定指数" | 被揭穿风险 |
| 舆论施压 | 通过KOL系统制造舆论压力，迫使某国政策转向 | 消耗公信力 |
| 暗网操作 | 雇佣黑客攻击某国基础设施，制造"灾害" | 高成本，被抓风险 |
| 外交挑拨 | 通过"白鸽"角色在外交渠道制造误会和摩擦 | 角色限定 |
| 资本攻击 | 集中做空某国货币，制造经济危机假象 | 巨量资金+对手盘风险 |

#### 催化阈值

每个事件有一个隐藏的"催化进度条"：

```
"暗棘入侵"催化进度:
  ├── 基础: 0%
  ├── 暗棘与他国关系恶化: +20%
  ├── 关于暗棘威胁的假新闻传播: +15%
  ├── 暗棘军工股被大量买入: +10%
  ├── 暗棘国内经济恶化: +10%
  └── 玩家暗中操作（暗网/外交）: +5~20%

  当进度达到100% → 事件触发
```

玩家可以加速这个进度条——也可以通过反向操作来减缓它。

### 9.4 连锁反应系统

事件发生后，不只是影响一个市场，而是通过连锁链传导：

```
连锁反应示例:

"焰砂减产石油":
  石油价格 +15%
  → EMK汇率 +8%（焰砂货币升值）
  → 铸链制造业成本上升 → 铸链股票 -5%
  → 碧芒新能源需求上升 → 碧芒新能源股 +7%
  → 全球通胀预期上升 → 龙头金 +3%
  → 伽蓝可能加息预期 → KAL +2%
  → 市场情绪 -10（能源恐慌）
```

#### 连锁传导规则

```
每条连锁链有:
├── 起源事件（触发源）
├── 传导节点（受影响的资产/市场）
├── 传导系数（每个节点的影响力，0~1）
├── 传导延迟（从起源到该节点的时间）
└── 衰减率（每经过一个节点，影响衰减20-40%）
```

#### 连锁崩盘（高潮时刻）

当多个负面连锁链叠加时，可能触发连锁崩盘：

```
连锁崩盘条件:
  同时满足以下2项以上:
  ├── 全球情绪 < 20
  ├── 2个以上国家进入冲突层级3+
  ├── 任意主要货币单分钟跌幅 > 5%
  ├── 龙头金单分钟涨幅 > 8%
  └── AI恐慌性抛售量 > 正常水平的5倍

连锁崩盘效果:
  ├── 所有风险资产额外 -10%~-20%
  ├── 龙头金额外 +15%~+25%
  ├── 市场进入"熔断模式"，每10秒才能交易一次
  ├── AI进入纯恐慌模式（只有卖出，没有买入）
  └── 持续30-60秒后开始恢复（抄底资金入场）
```

### 9.5 战争系统详解

#### 战争进程

```
阶段1: 谍报期（30-60秒）
  → 暗棘军事活动增加
  → 全球情报网络开始预警
  → 市场开始不安，军工股微涨
  → 玩家可以通过"信息差交易"提前获知

阶段2: 摩擦期（20-40秒）
  → 边境冲突、军事演习
  → 外交抗议升级
  → 市场开始波动，避险资产上涨
  → 玩家可以通过外交/舆论阻止或加速

阶段3: 爆发期（瞬间）
  → 正式开战
  → 市场瞬间剧烈波动
  → 全球情绪暴跌30-50点
  → 所有风险资产暴跌10-20%
  → 军工股暴涨15-30%
  → 龙头金暴涨20-30%
  → 石油暴涨20-40%

阶段4: 战争期（持续1-3分钟）
  → 市场在低位剧烈震荡
  → 每隔15-30秒触发一次"战况更新"（胜负消息）
  → 胜利消息→短暂反弹 / 失败消息→继续下跌
  → 玩家可以在战况更新之间做短线交易

阶段5: 结局
  ├── 一方获胜 → 战败国资产崩盘，战胜国资产反弹
  ├── 停火协议 → 市场反弹，但不会回到战前水平
  ├── 陷入僵局 → 市场持续低迷
  └── 升级为全面战争 → 触发L5终局事件
```

#### 战争对各国的影响矩阵

```
战争参与者: 直接影响（最大）
  → 本国货币暴跌20-50%
  → 本国股市暂停或暴跌30-60%
  → 国债收益率飙升

邻国: 间接影响（中等）
  → 难民涌入、贸易中断
  → 货币贬值5-15%
  → 军工股上涨

大国（伽蓝等）: 投机影响
  → 军工股涨10-20%
  → 可能"选边站"引发更多不确定性
  → 避险资产涨

全球市场: 情绪影响
  → 恐慌情绪蔓延
  → 龙头金、永恒等避险资产涨
  → 高风险资产（虚拟币等）跌
```

## 十、KOL系统（玩家作为KOL）

### 10.1 玩家KOL属性

```
玩家KOL面板:
├── 粉丝数（核心资源）
│   ├── 影响市场操控范围
│   ├── 粉丝会自发跟风交易
│   └── 粉丝越多，一句话的影响力越大
├── 公信力（0-100）
│   ├── 影响粉丝的跟随意愿
│   ├── 预测准确 → 公信力+5
│   ├── 被揭穿撒谎 → 公信力-15~-25
│   └── 公信力<20 → 粉丝开始大量流失
├── 热度（0-100）
│   ├── 影响新粉丝增长速度
│   ├── 发 controversial 内容 → 热度+10
│   ├── 长时间不发内容 → 热度自然衰减
│   └── 热度高 → 新粉丝涌入
├── 各平台账号
│   ├── 全球社交平台（类似Twitter）
│   ├── 金融圈论坛
│   ├── 视频直播平台
│   └── 暗网论坛
└── 影响力评级（S/A/B/C/D）
    └── 综合粉丝+公信力+热度的等级
```

### 10.3 粉丝系统

#### 粉丝类型

```
粉丝构成:
├── 铁粉（20%）
│   ├── 无脑跟随，几乎不质疑
│   ├── 交易量 = 总资金×80%
│   └── 长期信任，即使预测错误也不离场
├── 普通粉（50%）
│   ├── 跟随但有判断力
│   ├── 交易量 = 总资金×40%
│   └── 连续2次预测失败后开始动摇
├── 路人粉（20%）
│   ├── 偶尔关注，不完全跟随
│   ├── 交易量 = 总资金×15%
│   └── 一次预测失败就可能脱粉
└── 黑粉/反串（10%）
    ├── 关注是为了反驳你
    ├── 不会跟随交易，反而可能反向操作
    └── 会积极拆穿你的假消息
```

#### 粉丝增长/流失

```
增长:
├── 预测准确 → +5%~15%粉丝
├── 大赚喊单成功 → +20%~40%粉丝（一夜涨粉）
├── 制造争议话题 → +10%粉丝（关注度上升）
├── 攻击其他KOL成功 → 对方粉丝10%流入
└── 自然增长（热度>60时）→ 每tick +0.5%

流失:
├── 预测错误 → -5%~10%
├── 被揭穿撒谎 → -20%~50%
├── 长期不发内容 → -2%/分钟
├── 公信力跌破20 → 铁粉外全部流失
└── 被其他KOL击败 → 流向胜者
```

### 10.4 KOL对战

玩家可以与其他AI KOL进行舆论战：

```
对战流程:
1. 玩家发推看多某资产
2. AI KOL发推反驳看空
3. 粉丝分成两派，市场出现分歧
4. 最终价格走向决定胜负
5. 胜者 → 公信力+20，获得对方部分粉丝
6. 败者 → 公信力-15，粉丝流失
```

│ 设计"白皮书"│   │ 预留筹码  │    │ KOL喊单  │    │ 制造FOMO  │    │ 收割离场  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### 11.2 设计阶段

```
币的属性:
├── 名称（玩家自定）
├── 叙事/故事（为什么要买这个币？）
│   ├── 预设模板：
│   │   - "AI驱动的去中心化金融协议"
│   │   - "全球首个碳中和虚拟货币"
│   │   - "暗网匿名支付解决方案"
│   │   - "Gamified社交代币"
│   │   - "有军方背景的秘密项目"
│   └── 自定义文本
├── 供应量（1百万 ~ 100亿）
├── 初始价格
├── 预留筹码比例（玩家自己保留多少比例）
└── Rug机制设定
    ├── 手动触发（玩家自己决定什么时候跑）
    └── 自动触发（设定条件，如价格达到X时自动Rug）
```

---

### 11.3 发行阶段

```
发行选项:
├── 发行平台: 只能在汐潮自由邦的交易所发行
├── 流动性池: 玩家需要注入初始流动性（自己掏钱）
├── 预留筹码: 发行时自动分配给玩家设定的比例
└── 上架费: 少量资金
```

---

### 11.4 营销+拉盘阶段

```
营销手段:
├── AI水军（在社交媒体上刷帖造势）
├── 假新闻（制造关于这个币的利好新闻）
├── KOL喊单（花钱请KOL推荐，或用自己的KOL账号）
├── "白皮书"发布（增加可信度）
├── 空投承诺（承诺持有者会获得额外奖励）
├── 自己用资金买入推高价格（制造上涨假象）
└── 社区建设（创建"粉丝群"，制造归属感）

拉盘机制:
├── 玩家自己买入 → 价格上涨
├── 预留筹码的30%用于做市（挂买单制造需求假象）
└── 关键节点：当币价突破某个心理价位时，AI韭菜会自动FOMO入场
```

---

### 11.5 跑路阶段

```
Rug Pull执行:
├── 玩家点击"抽走流动性"
├── 所有流动性池资金瞬间转入玩家钱包
├── 所有持有者（包括AI韭菜）的持仓瞬间归零
├── 币价归零
└── 触发后果

最佳时机指标:
├── 市值已经达到目标（玩家自定义目标）
├── 买入势头开始减弱（新买入量下降）
├── 有人开始质疑（AI开始检测到异常）
├── 有其他鲸鱼AI开始卖出（可能提前跑路）
└── 贪婪指数达到峰值（韭菜最疯狂的时候）
```

---

### 11.6 Rug Pull的后果

```
后果（按严重程度递增）:
├── 公信力暴跌 -30~-60
├── 粉丝大量流失（-50%~-80%）
├── 被AI KOL公开谴责
├── 后续发币的可信度大幅降低（AI会记住）
├── 被多国监管列入黑名单
├── 在非汐潮地区可能被冻结资产
└── 多次Rug后，AI形成"群体记忆"，几乎不可能再骗到人

降低后果的方法:
├── 在汐潮自由邦操作（法律后果=0）
├── 使用暗影操盘手的"不留痕迹"技能
├── 嫁祸给其他AI或玩家
└── 制造"不是Rug，是被黑客攻击"的假新闻
```

## 十三、Roguelike结构

### 13.1 单局结构

```
一局游戏:
├── 选角色
├── 选起始地区
├── 随机生成:
│   ├── 初始市场状态（各国经济状况、关系网络）
│   ├── 事件池排列（本局会出现哪些事件）
│   ├── AI市场参与者（随机生成属性）
│   ├── 虚拟币初始池
│   └── 特殊规则（本局可能有的特殊modifier）
├── 游戏进行（15-45分钟）
│   ├── 市场实时运转
│   ├── 事件按计划触发（或被玩家催化/阻止）
│   └── 玩家操控市场获利
└── 结局判定
    ├── 主动结束：玩家自认赚够了，选择"金盆洗手"
    ├── 被动结束：
    │   ├── 资产归零（爆仓/被Rug）
    │   ├── 触发终局事件（全球崩溃）
    │   ├── 角色专属失败（白鸽任务失败、暗影被抓等）
    │   └── 时间到（每局有最大时间限制）
    └── 评分：
        ├── 最终资产
        ├── 操作评分（操控了多大行情）
        ├── 存活时间
        └── 成就完成度
```

---

### 13.2 局外成长（Meta Progression）

每局结束后，根据评分获得Meta点数，用于解锁永久内容：

```
解锁内容:
├── 新角色
│   ├── 初始只有2-3个角色可选
│   └── 通过特定成就解锁其他角色
├── 新新闻模板
│   ├── 更多类型和更高级别的假新闻模板
│   └── 解锁后所有角色可用
├── 新事件
│   ├── 向事件池中添加新事件
│   └── 增加游戏的多样性和不可预测性
├── 新币种模板
│   ├── 更有创意的"发币叙事"
│   └── 更好的Rug机制
├── 起始加成
│   ├── 额外初始资金（+5%~+20%）
│   ├── 起始粉丝加成
│   └── 起始关系网加成
├── 难度modifier
│   ├── 解锁更高难度（极端波动、更多黑天鹅等）
│   └── 高难度 = 更高Meta点数奖励
└── 特殊规则
    ├── "所有国家都处于战争边缘"
    ├── "虚拟币是唯一的交易品种"
    ├── "只有假新闻，没有真新闻"
    └── 自定义规则组合
```

---

### 13.3 每局的特殊Modifier

每局开始时可能随机附带一个特殊规则，增加重玩价值：

| Modifier | 效果 |
|----------|------|
| **全球牛市** | 所有资产情绪基准从50提升到65 |
| **末日倒计时** | 10分钟后必触发一次L5终局事件 |
| **AI叛乱** | AI交易者变得更理性，更难被骗 |
| **信息混沌** | 所有新闻可信度随机±30% |
| **监管风暴** | 汐潮不再是法外之地，假新闻/操纵被严厉打击 |
| **资源诅咒** | 大宗商品波动率×3，焰砂成为全场焦点 |
| **虚拟币狂热** | 虚拟币市场容量×5，传统市场资金外流 |
| **和平年代** | 战争不会发生，但经济泡沫更容易破裂 |
| **暗战模式** | 所有国家关系-30，冷战成为常态 |

## 用户偏好记录

- 希望接近真实期货/股票市场但有游玩性
- 不想映射真实国家（避免政治麻烦）
- 国家要有深度模拟
- 市场驱动因素要像真实世界一样多元（政策、汇率、技术、新闻、战争、资金流等）
- 避险资产不绑定任何国家（龙头金、永恒）
