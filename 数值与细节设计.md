## 一、数值平衡

### 设计基准

```
核心设计目标:
├── 一局20分钟（标准模式），优秀玩家资产增长10-50倍
├── 高风险操作（杠杆/Rug）可以获得100-1000倍，但归零概率也高
├── "什么都不做"的旁观者大约亏20-40%（通胀+手续费吃掉）
└── 市场总流动性需要足够大，玩家不能轻易单方面控盘
```

## 二、游戏模式

| 模式 | 时长 | 节奏 | 事件频率 | 说明 |
|------|------|------|----------|------|
| **闪电战** | 10分钟 | 极快 | 黑天鹅每2分钟 | 高频决策，适合短时间玩一把 |
| **标准局** | 20分钟 | 适中 | 黑天鹅每5分钟 | 默认模式，完整体验 |
| **持久战** | 45分钟 | 慢热 | 黑天鹅每10分钟 | 可以慢慢布局，深度操控 |
| **沙盒模式** | 无限制 | 自定义 | 自定义 | 自由调整所有参数 |

不同模式下Tick间隔按比例缩放：
```
闪电战: 所有Tick间隔 ×0.5
标准局: 默认间隔
持久战: 所有Tick间隔 ×1.5
沙盒:   玩家自定
```

## 三、速度控制

```
控制选项:
├── 暂停: 所有引擎停止Tick，UI可操作（查看数据但不能交易）
├── 1x:  默认速度
├── 2x:  所有Tick间隔减半，AI计算频率翻倍
├── 3x:  所有Tick间隔÷3
└── 最大速度: 引擎尽可能快跑（用于沙盒跳过无聊阶段）

实现方式:
├── GameLoop的tickIntervals除以speedMultiplier
├── AI分帧的batchSize随速度乘以speedMultiplier
└── 暂停时GameLoop.update()直接return
```

## 四、新手引导

```
渐进式解锁（以标准局20分钟为例）:

0:00 - 开局: 只能看UI、买卖操作
        UI灰掉的部分: 假新闻编辑器、KOL面板、发币
        顶部提示: "先用买卖熟悉市场"

2:00 - 解锁: 新闻/舆论流可见，假新闻编辑器
        提示: "试试制造一条假新闻影响市场"

5:00 - 解锁: KOL面板
        提示: "你现在可以用影响力操控市场了"

8:00 - 解锁: 全部功能
        提示: "所有工具已解锁，发挥你的想象力"
```


### 7.2 AI KOL自发互动

```
KOL之间:
├── 看多派 vs 看空派 → 自动发生辩论
├── 阴谋论者 → 会转发和夸大任何可疑信息
├── 攻击型 → 会主动找其他KOL的弱点攻击
├── 合作型 → 会与其他KOL互相推荐
└── 每隔15-30秒，AI KOL会自动发一条推文/观点

KOL与市场:
├── KOL会根据自己的持仓方向喊单
├── 亏钱的KOL公信力下降
├── 连续正确预测的KOL粉丝暴涨
└── 粉丝量超过1M的KOL获得"超级KOL"标签，市场影响力翻倍
```

### 7.3 KOL与玩家互动

```
关系值 -100 ~ +100:

提升关系:
├── 跟随他们的观点交易（+5）
├── 在公开场合赞美他们（+10）
├── 帮他们辟谣/辩护（+15）
├── 给他们分成/贿赂（+20，但对方如果是正直型会拒绝）
└── 结盟成功（+30）

降低关系:
├── 公开质疑他们（-10）
├── 和他们做对手盘（-5）
├── 揭穿他们的假消息（-20，如果对方在撒谎）
├── 抢他们的粉丝（-15）
└── Rug Pull坑了他们的粉丝（-30）

不同关系的互动:
├── 关系>+50: 会主动帮玩家喊单、分享内幕、联合攻击敌人
├── 关系+20~+50: 偶尔合作，不一定每次都帮
├── 关系-20~+20: 中立，按自己利益行事
├── 关系-50~-20: 会质疑玩家的观点、提醒粉丝小心
└── 关系<-50: 主动攻击玩家、号召粉丝反向操作
```

## 八、边缘情况处理

| 情况 | 处理方案 |
|------|----------|
| 玩家资金归零但有持仓 | 不算出局，持仓也归零（爆仓）则出局。持仓还在浮盈中可以等回本 |
| 所有国家同时开战 | 触发L5终局事件"世界大战"，所有资产进入极端模式，30秒后强制结算 |
| 玩家挂机什么都不做 | 市场正常运行，玩家资产随市场波动。20分钟后正常结算，大概率亏钱 |
| 玩家反复发同一条假新闻 | AI有"免疫记忆"：同模板同目标使用3次后效果降为10%，5次后完全免疫 |
| 玩家同时操控多个系统 | 允许并发操作，但直播/AMA期间不能做其他事（强制专注） |
| 市场流动性归零 | 资产暂停交易，等待央行/鲸鱼注入流动性。玩家挂单留在队列中 |
| 玩家做空后价格无限上涨 | 杠杆爆仓机制保护。无杠杆做空最多亏光本金，有杠杆按保证金比例爆仓 |
| KOL公信力降到0 | 失去所有粉丝，所有KOL操作失效。需从0重建（非常困难） |
| 骗子AI的币没人买 | 骗子AI加大营销力度，持续失败则放弃换下一个 |
| 玩家被所有国家通缉 | 资产在非汐潮地区被冻结，只能在汐潮操作。可用暗影"消失"技能重置 |
| 一局内Rug Pull超过3次 | 触发"信用破产"，AI形成群体记忆，后续发币/新闻效果永久降为10% |
| 暗棘黑市汇率与官方差距>20倍 | 触发"暗棘开放"事件概率大幅提升，可能重置汇率体系 |
| 碧芒同时被3个以上国家施压 | 触发"种子库封锁"事件，全球粮食恐慌 |
